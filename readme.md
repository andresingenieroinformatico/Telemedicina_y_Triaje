# Telemedicina y Triaje — Microservicios

Este repositorio está organizado como proyecto de microservicios con tres servicios separados:

- `microservices/agendamiento/` — servicio de agendamiento y consultas médicas.
- `microservices/triage/` — servicio de triaje automatizado.
- `microservices/usuarios/` — servicio de usuarios y autenticación.

## Estructura

```text
microservices/
  agendamiento/
  triage/
  usuarios/
```

Cada servicio conserva su propia aplicación, dependencias y documentación.
